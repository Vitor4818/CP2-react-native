## Instalação

1. **Clone o repositório**:

```bash
git clone https://github.com/Vitor4818/CP2-react-native.git
```

2. **Instale as dependências**:

```bash
npm install
npm install @react-navigation/native-stack
npm install @react-navigation/bottom-tabs
npm install expo
```

3. **Rode o projeto**:

```bash
npm start
npx expo start
```
